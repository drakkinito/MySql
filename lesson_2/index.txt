DDL - Data Definition Language (е сімейство комп'ютерних мов, що використовуються в комп'ютерних програмах або користувачами баз даних для опису структури даних.)

DDL мають свою функціональну здатність, організовану за початковим словом в заяві (запит), яке майже завжди є дієсловом.

Create (Створити БД)
Alter (Змінити БД)
Drop (Видалити БД)


ALTER TABLE clients auto_increment = 0; -- повертає автоінкремент до 0
ALTER TABLE cars auto_increment = 0; -- повертає автоінкремент до 0
